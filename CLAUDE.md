# SkyLife Aircrafts — Claude Code Instructions

## About this repo
Public website for SkyLife Aircrafts, a private aircraft brokerage 
based in Willowdale, Ontario.

## Rules
- Main file is index.html — never delete or rename it
- vercel.json handles Vercel deployment — never modify it
- After EVERY file change: git add . && git commit && git push origin main
- Always confirm what was pushed after each operation

## Brand
- Navy: #0a1628 | Teal: #3ecfb2
- Logo: transparent PNG embedded as base64 in index.html

## Workflow
When given a new index.html file, replace the existing one and push immediately.
